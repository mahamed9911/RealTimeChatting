# RealTimeChatting

This is a real time chatting web application that required the user to create an account before being able to chat with other users


technologies used 

**Client-side**

  react.js
  
  
**Server-side**

  Node.js
  
  Express.js
  
  PostgreSQl
  
  Sequelize
  
  Bcrypt
  
  Socket.io
